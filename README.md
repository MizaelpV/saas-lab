# SaaS Lab

Monorepo para el desarrollo de aplicaciones SaaS.

## Estructura del Proyecto

```
/saas-lab
  /apps
    /web      - Frontend (React + Vite/Next)
    /api      - Backend API (FastAPI o NestJS)
    /agents   - Agentes de IA (LangGraph)
  /infrastructure
    /scripts  - Scripts de infraestructura y deployment
```

## Apps

### Web
Frontend desarrollado con React y Vite/Next.js

### API
Backend API desarrollado con FastAPI o NestJS

### Agents
Agentes de IA desarrollados con LangGraph
